## Rationale

1. **ATR labeling adapts to SOL's higher volatility.** SOL NATR ~5-8% means static 8% TP is often too close and 4% SL hits from noise. With ATR labeling at 3.5x/1.75x, barriers scale: if SOL NATR is 6%, TP = 21%, SL = 10.5%. This gives the model more room.

2. **46 explicit features fix the samples/feature ratio.** Iter 123 had 185 features with ~2,200 training samples (ratio 12). With 46 features, ratio = 48 — close to the 50 target. Using the same feature categories as the meme model (which went from 67→45→OOS doubled).

3. **Meme model architecture is proven.** DOGE+SHIB with this exact config (ATR labeling, pruned features, 5-seed ensemble) produced OOS Sharpe +0.73. SOL has stronger fundamentals (higher volume, more institutional flow).
